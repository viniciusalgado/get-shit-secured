# Map Codebase

**Placeholder workflow** - Full implementation coming soon.

## Description

Analyzes the codebase to identify components, data flows, and security boundaries. This provides the foundational context for all other security workflows.

## Activities

- Identify application components and modules
- Map data flow between components
- Identify authentication/authorization boundaries
- Catalog external dependencies
- Document API endpoints and access controls

## Output

- Codebase inventory
- Data flow diagrams
- Component interaction map
- Security boundary identification
